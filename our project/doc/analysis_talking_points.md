# 答辩口播稿要点

本文档用于系统展示、答辩陈述和演示讲解，口径与当前真实数据保持一致。

## 一、8 张图分别说明什么

1. 学习时长分布图
说明全体学生学习投入的总体分布，观察是否存在明显的高投入和低投入分层。

2. 图书馆访问次数分布图
说明线下学习场景的活跃程度，用来判断学生线下自习与学习习惯的差异。

3. 健康指数分布图
说明学生在运动、体测和健康发展方面的整体状态，体现学业之外的身心发展情况。

4. 夜间上网比例分布图
说明学生行为规律差异，夜间上网比例越高，往往意味着作息节奏波动越明显。

5. 学习时长 vs 风险概率
说明学习投入与风险水平之间的关系，便于观察低投入是否更容易伴随高风险。

6. 图书馆访问 vs 风险概率
说明线下学习活跃度与风险水平之间的关系，辅助识别学习场景稳定性不足的学生。

7. 各聚类类别人数分布
说明当前学生群体可以被划分成哪些典型类型，以及每一类学生的规模占比。

8. 不同聚类的核心特征均值对比图
说明不同学生群体在学习投入、自律、健康、发展和行为规律上的差异，是群体画像的核心支撑图。

## 二、5 维画像分别代表什么

1. 学习投入
对应学习指数，综合反映学习时长、课程学习、学习参与和线下学习投入情况。

2. 自律水平
对应自律指数，主要反映学生日常行为规律性和执行稳定性。

3. 健康发展
对应健康指数，主要反映运动、体测和身体状态相关表现。

4. 综合发展
对应综合发展指数，主要反映综合测评、活动参与和整体成长表现。

5. 风险反向值
对应风险概率的反向表达，值越高表示风险越低，值越低表示风险越需要关注。

## 三、风险解释 3 个维度分别是什么

1. 学习投入风险
由学习指数、学习时长和图书馆活跃度共同解释。
口播建议：
如果学习投入不足，我们会认为学生在学业持续性上存在潜在风险。

2. 行为规律风险
由自律指数和夜间上网比例共同解释。
口播建议：
如果夜间上网偏高、自律指数偏低，通常意味着作息和行为节奏存在波动。

3. 健康发展风险
由健康指数和综合发展指数共同解释。
口播建议：
如果健康发展偏弱，说明学生可能在运动投入、身体状态或整体成长参与上存在短板。

## 四、群体标签怎么讲

当前系统使用的是展示友好的群体名称，而不是“第 0 类、第 1 类”：

- 高投入稳健型
- 低投入风险型
- 夜间波动型
- 均衡发展型
- 发展过渡型

口播建议：
系统不是只告诉我们学生属于第几类，而是把每一类学生转译为可理解、可展示、可干预的群体标签。

## 五、报告页怎么讲

口播顺序建议：

1. 先说明该学生的基础身份信息
2. 再说明该学生的 5 维个体画像
3. 然后说明其风险等级与所属群体类型
4. 接着解释风险来自哪几个维度
5. 最后给出面向辅导员或学生本人的个性化建议

一句话总结模板：
本系统不是只输出一个风险标签，而是把学生的学习投入、行为规律、健康发展和群体归属整合起来，形成可解释、可展示、可干预的个性化分析结果。
