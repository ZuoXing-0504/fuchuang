# 知行雷达可运行 Demo 说明

## Demo 运行内容

- 后端 Flask 服务
- 管理端 Web 页面
- 学生端 Web 页面
- 安卓移动端安装包
- 模型文件、图表文件与数据库文件

## 运行方式

- 执行 Bootstrap-Dev.ps1 初始化环境
- 执行 Start-Dev.ps1 启动前后端
- 管理端访问 localhost:3200，学生端访问 localhost:3201，后端访问 localhost:5000
- 安卓端可通过最新打包的 apk 安装体验
